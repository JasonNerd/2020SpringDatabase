【使用方法】
1、安装requirenments下的库，注意最好使用conda创建虚拟环境，python=3.6 or python=3.7
在虚拟环境下使用pip命令安装依赖
2、推荐使用pycharm指定刚刚创建的虚拟环境，在激活虚拟环境的情况下
	A、运行 flask init-db --password=xxxxx
	其中xxxxx是你自己的mysql server的密码
	B、经过上面一步的操作，此时server端会有一个hitblog数据库，内含建好的表，此时
	键入	set FLASK_APP=HITBlog
	C、flask run
注意，2中工作目录应为HitBlog同目录下，不要进到HitBlog内

【2021-04-18】已完成基本功能
